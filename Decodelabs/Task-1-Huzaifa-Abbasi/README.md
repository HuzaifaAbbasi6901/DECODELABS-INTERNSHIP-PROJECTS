# Project 1 – Static Website Hosting on Azure

## Objective
Host a static website using Microsoft Azure Storage.

## Azure Services Used
- Azure Storage Account
- Static Website Hosting

## Steps Performed
1. Created a Storage Account.
2. Enabled Static Website Hosting.
3. Uploaded index.html and style.css.
4. Verified the website using the endpoint.

## Outcome
The website was successfully hosted and accessible through the Azure static website endpoint.
