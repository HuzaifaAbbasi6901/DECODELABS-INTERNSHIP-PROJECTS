# Project 3 – Advanced Azure Cloud Deployment

## Overview

This project focuses on deploying and managing cloud infrastructure using Microsoft Azure. It demonstrates the practical implementation of Azure services, resource management, and deployment verification as part of the Decode Labs Cloud Computing Internship.

## Objective

* Deploy and configure Azure cloud resources.
* Verify successful deployment and functionality.
* Gain hands-on experience with Microsoft Azure services.
* Document the deployment process and results.

## Azure Services Used

* Microsoft Azure Portal
* Azure Resource Group
* Azure Virtual Machine
* Azure Storage Account
* Virtual Network (VNet)
* Network Security Group (NSG)
* Public IP Address

> **Note:** Include only the Azure services that were actually used in your Project 3.

## Implementation Steps

1. Created or selected the required Azure Resource Group.
2. Configured the necessary Azure resources.
3. Applied the required networking and security settings.
4. Completed the deployment process.
5. Verified that all resources were functioning correctly.
6. Captured screenshots as proof of successful completion.

## Project Outcome

The required Azure resources were successfully deployed and verified. This project strengthened practical skills in cloud infrastructure deployment, Azure resource management, networking, and deployment verification.
