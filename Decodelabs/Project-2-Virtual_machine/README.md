# Project 2 – Azure Virtual Machine Deployment

## Overview

This project demonstrates the deployment and basic configuration of a Microsoft Azure Virtual Machine (VM). The objective was to create a secure virtual machine, configure its networking components, and verify successful deployment through the Azure Portal.

## Objective

* Create a Resource Group.
* Deploy a Virtual Machine on Microsoft Azure.
* Configure networking components, including a Virtual Network, Public IP Address, and Network Security Group (NSG).
* Verify that the virtual machine is running successfully.

## Azure Services Used

* Microsoft Azure Portal
* Azure Virtual Machine (VM)
* Resource Group
* Virtual Network (VNet)
* Network Security Group (NSG)
* Public IP Address

## Implementation Steps

1. Created a new Resource Group.
2. Deployed a Virtual Machine with the required configuration.
3. Configured the Virtual Network and Subnet.
4. Assigned a Public IP Address to the VM.
5. Configured the Network Security Group with the required inbound rules.
6. Verified that the Virtual Machine was successfully created and running.
7. Captured screenshots as evidence of successful deployment.

## Project Outcome

The Azure Virtual Machine was successfully deployed and configured. All required networking resources were created, and the VM was verified as operational through the Azure Portal.
